2. Quirks mode - режим умісності, що використовується браузерами для відображення застарілих веб-сторінок (створених ще до початку широкого застосування веб стандартів). В цьому режимі сучасні браузери при відображенні сторінки ведуть себе як їх старі версії, наприклад IE імітує роботу IE5, Mozzila - Netscape 4.

3. <q> - inline тег призначений для виділення цитати безпосередньо в тексті. Відображається як текст в подвійних лапках.
<cite> - inline тег призначений для виділення назви роботи, автора чи посилання на джерело цитування. Відображається курсивом.

4. <b> - inline тег призначений для візуального виділення тексту жирним шрифтом.
<strong> -  семантичний inline тег призначений для виділення важливого фрагменту тексту, на який потрібно звернути увагу користувача, сильне акцентування. Візуально виглядає так само як <b>.

5. <i> - inline тег призначений для візуального виділення тексту курсивом.
<em> -  семантичний inline тег призначений для акцентування уваги на фрагменті тексту. Візуально виглядає так само як <i>.
